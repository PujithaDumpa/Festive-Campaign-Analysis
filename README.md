# Festive-Campaign-Analysis
Festive campaign analysis for Atliq Mart (Diwali 2023 &amp; Sankranti 2024) using Excel, SQL, and Power BI.

## Project Overview 
Festive seasons significantly influence sales performance in the retail industry, and promotional campaigns play a key role in driving customer purchases during these periods.This project evaluates the effectiveness of festive promotional campaigns using sales data, focusing on how promotion types, product categories, and regions impact incremental revenue and sales uplift. By analyzing key performance metrics, the project provides insights that support data-driven decision-making for future festive campaigns.

## Business Problem
Festive campaigns involve significant investment in discounts and promotions, but their impact on revenue and sales uplift can vary widely across products, categories, and regions. Without proper analysis, businesses risk allocating resources to ineffective promotions.

This project aims to analyze festive sales data to answer key business questions such as:
- Which promotion types generate the highest incremental revenue?
- Which product categories and cities respond best to festive campaigns?
- Are there promotions that result in negative or low returns?

The goal is to use data analysis to support better-informed promotional and campaign planning decisions.

## Data Source
The analysis is based on data obtained from AtliQ Mart's internal databases. The main datasets used include fact_events, dim_products, dim_stores, and dim_campaigns. These datasets contain information about product sales, store locations, promotional events, and campaign revenues.
